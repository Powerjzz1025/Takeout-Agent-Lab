9. 长段落

这是一段较长的中文说明，用来观察右侧预览器是否能保持清晰的阅读节奏。一个好的文档预览应该避免所有内容都贴在白底上没有层次，也应该让代码块、引用块、表格和普通段落之间有足够的视觉差异。如果 .md 预览器的样式缺失，通常会看到所有模块都变成接近白底，只剩文字本身。
